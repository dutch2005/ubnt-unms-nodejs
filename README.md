# UNMS Server

## Technologies
* __Application server__ - [Hapi](http://hapijs.com/)
* __Database__ - [PostgreSQL](https://www.postgresql.com/)
* __Cache__ - [Redis](https://redis.io/)

## Deployment

### AWS
